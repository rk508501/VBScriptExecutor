Example VBScript Endpoint

This is an example endpoint that allows you to run VBScript code and get the output.

Endpoint

POST /run-vbscript

Request Parameters

  • script: The VBScript code to run. This should be passed in as a plain text body.

Example Request

    
POST /run-vbscript HTTP/1.1
Host: example.com
Content-Type: text/plain

Dim objExcel, objWorkbook
Set objExcel = CreateObject("Excel.Application")
Set objWorkbook = objExcel.Workbooks.Open("C:\path\to\file.xlsx")

' do something with the workbook
objWorkbook.Worksheets("Sheet1").Range("A1").Value = "Hello, World!"
objWorkbook.Save
objWorkbook.Close
objExcel.Quit

Set objWorkbook = Nothing
Set objExcel = Nothing

Example Response

    
{
  "success": true,
  "output": "Script executed successfully."
}

Notes

  • The Content-Type header should be set to text/plain.
  • The response will be in JSON format, with a success flag indicating whether the script executed successfully, and an output field containing the output of the script.
  • Make sure that your VBScript code does not contain any sensitive information, as it will be transmitted as plain text.
